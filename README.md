# Pixel Hover

**Live demo:** https://exzenter.github.io/pixel-hover/

This repository contains a simple HTML mockup demonstrating a pixel hover effect.

## Preview

![Pixel Hover – EXZENT Mockup](2026-03-07%2000_09_41-Pixel%20Hover%20%E2%80%93%20EXZENT%20Mockup.png)

## Overview

The pixel hover effect creates an interactive pixel grid that reacts to mouse movement, producing a dynamic, glowing pixel art-style animation. Built as a lightweight WordPress Gutenberg block plugin (`exzent-pixel-hover`), it can also be used standalone via `index.html`.

## Files

| File | Description |
|------|-------------|
| `index.html` | Standalone HTML demo |
| `exzent-pixel-hover/` | WordPress Gutenberg block plugin |
| `exzent-pixel-hover/exzent-pixel-hover.php` | Plugin entry point |
| `exzent-pixel-hover/block.js` | Gutenberg block registration |
| `exzent-pixel-hover/pixel-hover.js` | Core pixel hover logic |
| `exzent-pixel-hover/editor.css` | Block editor styles |
| `exzent-pixel-hover.zip` | Packaged plugin for WordPress upload |

## WordPress Plugin

To install the Gutenberg block plugin:

1. Download `exzent-pixel-hover.zip`
2. Go to **WordPress Admin → Plugins → Add New → Upload Plugin**
3. Upload the zip and activate
4. Add the **Pixel Hover** block in the Gutenberg editor
